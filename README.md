# Planter-Dividers
A modular 3D printable insert system for planter beds, allowing for easier use and transport.

**FEATURES**
- Full size (8" x 5.5" x 4.5") and half size (4" x 5.5" x 4.5") inserts.
- Full size and half size aeration trays (0.5" tall).
- 1/2" raised bed with 21 water drainage holes.
- 3 divets on each side to allow water movement.
- Hande that extends 1" above the top of the insert to allow for easy handling.

I chose to use ratios for the palcments of the water drainage holes, so resizing the insert will automatically replace the holes in a 3x7 grid for full size and 3x3 for half size. Created entirely using Onshape CAD software, and intended to be printed with PLA.
